# Classificacao-Doencas-Oculares-DL
Projeto de classificação de 4 doenças oculares via Deep Learning. Implementamos e comparamos 4 arquiteturas (CNN, VGG19, ResNet-50, Efficient Net). O trabalho avalia o impacto do Data Augmentation e a estabilidade dos modelos através de 5 execuções, reportando Acurácia, F1-score e Matriz de Confusão.
